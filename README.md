# EENT
Electron EcoNet Two

This repository contains a few files for the Econet interface for the Acorn Electron with a PLUS-1

file list:
* EENT building instructions - stardot.org.uk.pdf -> building instructions
* EENT-PartList.pdf -> part list
* EENT-rev1.1.pdf -> schematic diagram
* EENTv1mk2.EQN -> GAL source file
* EENTv1mk2.JED -> Binary file with GAL contents
* EENTv1-NFS3.35.rom.BIN -> Binary file with EEPROM (N28F010) contents, only NetFS for the Elk

For more information and support: https://stardot.org.uk/forums/viewtopic.php?p=474088
